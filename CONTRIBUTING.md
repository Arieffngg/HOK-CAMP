# 🤝 Panduan Kontribusi

Terima kasih telah tertarik untuk berkontribusi pada **HOK Camp Bot**! Kami sangat menghargai bantuan Anda untuk menjadikan bot ini pionir intelijen Honor of Kings.

## 📜 Aturan Main
1. **Arsitektur 5 Pilar:** Setiap penambahan fitur harus selaras dengan struktur 5 pilar yang sudah ada.
2. **Kualitas Kode:** Kami tidak menerima *spaghetti code*. Pastikan kode Anda rapi, terdokumentasi, dan mudah dipahami.
3. **Data Privacy:** Jangan pernah mengunggah file `.env` atau folder `data/` hasil *scraping* pribadi Anda ke dalam *Pull Request*.
4. **Alur Kontribusi:**
   - *Fork* repositori ini.
   - Buat *branch* fitur baru (`git checkout -b fitur/NamaFitur`).
   - Lakukan *Commit* perubahan Anda.
   - Ajukan *Pull Request* (PR) untuk kami tinjau.

Mari kita bangun komunitas HOK yang lebih cerdas bersama-sama!
